# DevLens 📊

Interactive developer trends dashboard built from Stack Overflow Developer Survey 2024 data.

**[→ Try it live](https://luciferankon.github.io/devlens)**

## What it shows

- **Languages** — most used, loved, and wanted, with 5-year trend lines
- **Salaries** — by role, language, and years of experience
- **Tools & Frameworks** — web frameworks, databases, cloud platforms, AI tool adoption
- **Work Style** — remote vs hybrid vs office, job satisfaction, regional breakdown
- **Roles** — developer types, education levels, company sizes

## Built with

- Chart.js 4.4 — interactive charts
- Vanilla JS — no framework, no dependencies beyond Chart.js
- GitHub Pages — static hosting

## Data

Data sourced from the [Stack Overflow Developer Survey 2024](https://survey.stackoverflow.co/2024/) — 65,437 respondents worldwide. Figures are representative/approximate for visualisation purposes.
